Sensor de proximidad






Este circuito consiste en el constante monitoreo por parte del sensor, en el momento que se detecte algún movimiento, lo registrara y activara el led, cuando no se registre movimiento comenzara la espera para por fin volver al estado de reposo

from gpiozero import LED
from gpiozero import MotionSensor

led = LED(17)
pir = MotionSensor(4)
led.off()

while True:
pir.wait_for_no_motion()
print("no hay movimiento")
led.off()
pir.wait_for_motion()

led.on()
print("movimiento detectado")
